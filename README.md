# FINTECH REFUND BANK - Professional Banking Dashboard & Landing Page

## 🏦 Overview

A high-fidelity, professional banking website for FINTECH REFUND BANK featuring a minimalist design philosophy with Apple-esque aesthetics. The site includes a stunning landing page and a fully functional banking dashboard with real-time refund notifications.

## ✨ Key Features

### Landing Page (`index.html`)
- **Hero Section**: Eye-catching introduction with "Client Refunds" and "Claims Processing" focus
- **Refund Balance Card**: Large, soft-shadow card showing available refund amount ($12,847.50) with "Claim Funds" button
- **Features Grid**: 6 professional features with icons:
  - Military-Grade Security
  - Instant Processing
  - Real-Time Tracking
  - 24/7 Support
  - Zero Hidden Fees
  - Global Coverage
- **Success Stories/Reviews Section**: 6 verified client reviews with 5-star ratings
- **Live Refunds Feed**: Real-time notifications showing names of people receiving refunds
- **Professional Footer**: Quick links and direct support channels
- **Floating Contact Widget**: Quick access to WhatsApp and Facebook Messenger support

### Dashboard (`dashboard.html`)
- **Sleek Side Navigation**: Modern sidebar with icon-based navigation
- **Dashboard Overview**:
  - 4 Key Stat Cards (Total Refunds, Available Balance, Active Claims, Success Rate)
  - Action Card with "Claim Funds Now" button
  - Recent Transactions List with icons and status indicators
- **Create New Bank Account**: Multi-step form for account creation
- **Transactions View**: Full transaction history with filtering and sorting
- **Claims Management**: 
  - Display all claims with status tracking
  - Progress bars for processing claims
  - Option to file new claims
- **Account Settings**: 
  - Profile information editing
  - Two-factor authentication toggle
  - Password management
  - Notification preferences

## 🎨 Design Philosophy

### Theme: "Perfect White" - Minimalist & Ultra-Modern
- **Color Palette**:
  - Primary White: `#FFFFFF`
  - Soft Background: `#F8F9FA`
  - Light Grey: `#E8EAED`
  - Primary Blue Accent: `#3B82F6`
  - Dark Text: `#1F2937`

- **Typography**:
  - Font: Apple system fonts (Segoe UI, Helvetica Neue, sans-serif)
  - Clean, highly readable sans-serif
  - Generous whitespace and breathing room

- **Visual Elements**:
  - Subtle light grey borders (1px)
  - Soft shadows (0 2px 8px rgba(0,0,0,0.06))
  - Smooth transitions and animations
  - Rounded corners (12px default, 8px medium, 4px small)

## 🚀 Getting Started

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge)
- No backend server required - all functionality is frontend-based

### Installation
1. Navigate to the project directory
2. Open `index.html` in your web browser for the landing page
3. Click "Dashboard" button to access `dashboard.html`

### File Structure
```
Refund-Website2/
├── index.html           # Landing page
├── dashboard.html       # Banking dashboard
├── styles.css          # Main styles for landing page
├── dashboard.css       # Dashboard-specific styles
├── script.js           # Landing page functionality
└── dashboard.js        # Dashboard functionality
```
